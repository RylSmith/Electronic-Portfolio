Drawer Counter App

Description

The purpose of this project was to develop an application to help closing managers close drawers easier and with less errors. This was the final of CIS 131 where we were told to pick a personal project that we wanted to make. I was a closing manager at the time and I really liked the idea of having an application to help me with the process so I decided to make it.

Technologies Used

  • C#
  • Windows Forms
  • Visual Studio 2022

Sample Code

This function takes all user input variables made from a previous function and multiplys the values by respective money values. It then creates a new variable for the drawer total and adds up all the values after being adjusted above.


            public double AddDrawer(Drawer drawer, int Drawer_Money)
            {
                drawer.OneHundred = drawer.OneHundred * 100;
                drawer.Fifty = drawer.Fifty * 50;
                drawer.Twenty = drawer.Twenty * 20;
                drawer.Ten = drawer.Ten * 10;
                drawer.Five = drawer.Five * 5;
                drawer.One = drawer.One * 1;

                drawer.Quarters = drawer.Quarters * 0.25;
                drawer.Dimes = drawer.Dimes * 0.10;
                drawer.Nickels = drawer.Nickels * 0.05;
                drawer.Pennies = drawer.Pennies * 0.01;

                drawer.Two = drawer.Two * 2;
                drawer.Dollar_Coins = drawer.Dollar_Coins * 1;
                drawer.Half_Dollars = drawer.Half_Dollars * 0.50;

                double Drawer_Total;

                Drawer_Total = drawer.OneHundred + drawer.Fifty + drawer.Twenty + drawer.Ten + drawer.Five + drawer.One + drawer.Quarters 
                + drawer.Dimes + drawer.Nickels + drawer.Pennies + drawer.Two + drawer.Dollar_Coins + drawer.Half_Dollars;

                return PullDrawer(Math.Round(Drawer_Total, 2), Drawer_Money);

            }

This function saves drawer information in the files of the application. This function works by using StreamWriter to write to an existing file or make a new one if none matchs the name. The data for the file is pulled from the rich text box on the main GUI. After the file is done writing the function closes the StreamWriter and closes the application. 

            private void btnSave_Click(object sender, EventArgs e)
            {
                StreamWriter fileWriter = new StreamWriter("DrawerData.txt");
                fileWriter.WriteLine(richTxtBTotals.Text);

                fileWriter.Close();
                Close();
