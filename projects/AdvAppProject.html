This function checks whether a user is authenticated by retrieving the current session from Supabase. If no active session exists, the user is redirected to the login page. This ensures that protected parts of the application are only accessible to authenticated users.

            async function requireAuth() {
                const { data: { session }, error } = await supabase.auth.getSession();
                if (error) console.error('getSession error:', error);
                if (!session) {
                navigate('/login');
                return null;
                }
                return session;
            }

This function formats a time string into a more readable 12-hour format. If no time value is provided, it returns an empty string to avoid errors or displaying invalid data. The function splits the input time into hours and minutes, then creates a Date object and sets those values accordingly.

            const formatTime = (timeStr) => {
                if (!timeStr) return '';
                const [hours, minutes] = timeStr.split(':');
                const d = new Date();
                d.setHours(parseInt(hours), parseInt(minutes));
                return d.toLocaleTimeString('en-US', {
                hour: 'numeric',
                minute: '2-digit',
                hour12: true });
            };
